# Minecraft 1.21.10 Module

This is the active implementation module for KoHs Player Optimizer.

## Build

```bash
./gradlew :versions:mc1_21_10:build
```

## Key Components

- Fabric + Java 21 client mod
- Modular feature manager for rendering, movement, placement, interaction, and combat visuals
- Mod Menu integration
- Cloth Config tabbed settings screen
- Compatibility toggles for Sodium, Lithium, and Reese's Sodium Options
